
A sports buddy web app written in laravel and livewire. The goal ? To intergrate and connect more and more athletes together.

Laravel Backend + Livewire and Flux Frontend.


<h2>**REQUIRED TO RUN** : The geographical locations REST API consumed comes from one of my repositories: </h2>
<a href="https://github.com/tfjordanp/locations-rest-api" width="640" height="640">Locations REST API, written in Node.JS and Express</a>
